# Chatbots Conversation Projects

A collection of chatbot conversation experiments using different LLMs and approaches.

## Projects

### 1. Two-Way Chat
Basic conversation between two AI chatbots.

### 2. Three-Way Chat ‚≠ê
Advanced 3-way conversation between three AI characters (Alex, Blake, and Charlie) using OpenAI and Ollama models.

**Features:**
- Uses 1 system prompt + 1 user prompt with full conversation history
- Combines OpenAI (GPT-4o-mini) and Ollama (Llama3.2, Llama3.2:1b) models
- Fun, engaging personalities for each character
- Random debate topics

See the individual project folders for setup and usage instructions.

## Tech Stack

- Python 3.11+
- OpenAI API
- Ollama (for local LLMs)
- Requests library

## Setup

Each project has its own `requirements.txt`. Install dependencies in each project folder:

```bash
cd three-way-chat
pip install -r requirements.txt
```
